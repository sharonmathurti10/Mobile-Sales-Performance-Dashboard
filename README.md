# Mobile-Sales-Performance-Dashboard
Interactive Power BI dashboard analyzing mobile sales KPIs and performance trends 
# Project Overview 
This project analyzes mobile sales performance using Power BI.
The dashboard provides insights into total revenue, unit sales, brand contribution, payment methods, and month-over-month growth. 
The goal is to help businesses understand sales trends and performance drivers. 
# Business Questions Addressed 
- What is the total revenue and total units sold?
- Which brand contributes the highest revenue?
- How is revenue changing month-over-month?
- Which payment method generates the highest revenue?
- Which day of the week drives maximum sales?
# Dataset Information 
The dataset includes:
- Total Rows : 3836
- Total Columns : 12
- Transaction ID
- Date (DD-MM-YYYY)
- Brand
- Units sold
- Price per unit
- City
- Payment Method
- Customer Ratings
# Key KPIs:
- Total Revenue
- Total Units Sold
- Average Selling Price
- Revenue per Transaction
- Month-over-Month Growth %
# Dashboard Features 
- Revenue and Units sold by month
- Brand Revenue % Contribution
- Revenue by Payment Method
- Revenue by Day Name
- Interactive Year-Month Slicer
# Key Insights 
- Apple and Samsung contribute the highest revenue share.
- Saturday generates the highest sales revenue.
- Debit Card and UPI are dominant payment methods.
- Revenue shows fluctuations across months with positive MoM Growth of 0.8%.
- Brand contribution is relatively balanced across major brands.
